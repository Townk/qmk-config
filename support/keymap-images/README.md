# Layer Image Generator (pre-alpha)

The image generator uses [typst](https://typst.app) to compile a YAML file
representing my keymap into nice images to visually show my keyboard layer
definitions.

The Svalboard "library" that defines how to draw Svalboard keys is the file
`svalboard-keys.typ`. The big image with all layers in one page is generated by
the `keymap-overview.typ` file, and individual layers are generated by the
`keymap-layers.typ` file.

To update the images in the docs run the following:

```sh
$ typst compile keymap-layers.typ --format=png "../../docs/keymap-{p}.png"
$ typst compile keymap-layers.typ "../../docs/keymap-{p}.png" \
>   --ppi=120 --font-path="./fonts" --ignore-system-fonts

$ typst compile keymap-overview.typ "../../docs/keymap-overview.png" \
>   --ppi=120 --font-path="./fonts" --ignore-system-fonts

```

Make sure you have `typst CLI` installed before running these commands, and run
these two commands from within the `support/keymap-images` directory.
